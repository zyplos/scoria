# Changelog

All notable changes to the "scoria" color theme will be documented in this file.

## [1.0.3] - 2025-12-27

- Some agent UIs should look nicer and not bright yellow now
- Readjusted diff editor colors for extensions who only use the background color tokens

## [1.0.2] - 2025-12-26 

- Background colors in the diff editor have been made darker and should be easier on the eyes for people who are constantly staring at diffs
- Status bar background color during debugging and remote sessions have been made slightly darker

## [1.0.1] - 2024-04-13 

- Added support for Project IDX

## [1.0.0] - 2024-04-13

- Initial release
